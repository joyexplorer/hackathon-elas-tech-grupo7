# 📊 Telecom Churn Analytics

Dashboard interativo para análise de churn (cancelamento) de clientes de telecomunicações.

## 🎯 Sobre o Projeto

Este projeto realiza uma análise completa de churn para uma empresa de telecomunicações, utilizando técnicas de análise de dados e machine learning para identificar padrões de cancelamento e gerar insights estratégicos.

### Principais Funcionalidades:

- ✅ Análise exploratória de dados (EDA)
- ✅ Dashboard interativo com Streamlit
- ✅ Filtros dinâmicos por contrato, serviço e pagamento
- ✅ Análise de KPIs e métricas financeiras
- ✅ Identificação de perfis de alto risco
- ✅ Recomendações estratégicas baseadas em dados

## 📊 KPIs Analisados

- Taxa de Churn: 26.6%
- Receita mensal total: R$ 455k
- Receita em risco: R$ 139k (30.5%)
- Ticket médio ativo: R$ 61.31
- Ticket médio churn: R$ 74.44

## 🚀 Como Executar

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/telecom-churn-analytics.git
cd telecom-churn-analytics

```
### 2.  Instale as dependências

pip install -r requirements.txt

### 3.  Execute o dashboard
streamlit run dashboard/app.py
