# Hackathon Elas Tech – Grupo 7
## Projeto: Sistema Inteligente de Previsão de Churn (Telco Customer Churn)

---

## Objetivo do Projeto
Desenvolver uma solução completa de análise de dados capaz de:

- Realizar limpeza e tratamento de dados
- Aplicar análise exploratória
- Construir consultas SQL estratégicas
- Desenvolver dashboards analíticos
- Criar modelo preditivo de churn
- Gerar insights para retenção de clientes

---

## Dataset Utilizado
Telco Customer Churn

---

## Tecnologias Utilizadas
- Python
- Pandas
- SQL / SQLite
- Power BI
- GitHub
